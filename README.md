📖 Sobre o Projeto
Este repositório contém um script prático detalhando o funcionamento dos operadores fundamentais no JavaScript. É ideal para quem está consolidando a base em Análise e Desenvolvimento de Sistemas.

 🧪 Conteúdo Técnico
Operadores Aritméticos:Soma, subtração, multiplicação, divisão e módulo (%)
Operadores Relacionais:Comparações de igualdade, diferença e magnitude (maior/menor).
Operadores Lógicos: Implementação de lógica booleana usando && (AND), || (OR) e ! (NOT).

🛠️ Como Executar
1. Tenha o Node.js instalado em sua máquina.
2. Execute o comando no terminal:
   
   node operadores.js

📖 About the Project
This repository contains a practical script detailing the operation of fundamental operators in JavaScript. It's ideal for those strengthening their foundation in Systems Analysis and Development.

🧪 Technical Content
Arithmetic Operators: Addition, subtraction, multiplication, division, and modulus (%).

Relational Operators: Equality, difference, and magnitude comparisons (greater/less than).

Logical Operators: Boolean logic implementation using && (AND), || (OR), and ! (NOT).

🛠️ How to Run
Ensure you have Node.js installed.

Run the following command in your terminal:
node operadores.js
